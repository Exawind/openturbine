Capabilities and Roadmap

This section documents a non-exhaustive list of current OpenTurbine capabilities and roadmap for future capabilities.

Tip

If your project relies on a capability that is not yet present in OpenTurbine, please create an issue on the code project page.

Please acknowledge as a publication co-author any developer that has significantly contributed to implementing or improving specific capability that was used for that publication.

Tip
