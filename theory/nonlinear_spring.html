R3-based geometrically nonlinear, constitutively linear springïƒ

Overview: Geometrically nonlinear and constitutively linear spring defined in \(\underline{\underline{R}}^3\)

Spring Model Input at Initialization:

  • \(\underline{x}_1^0 \in \mathbb{R}^3\): node 1 reference position; inertial CS

  • \(\underline{x}_2^0 \in \mathbb{R}^3\): node 2 reference position; inertial CS

  • \(\underline{u}_1^i \in \mathbb{R}^3\): node 1 initial displacement

  • \(\underline{u}_2^i \in \mathbb{R}^3\): node 2 initial displacement

  • \(k\) constitutively linear spring stiffness

  • \(\ell^\mathrm{us}\) unstretched reference length

Degrees of Freedom:

  • \(\underline{u}_1 \in \mathbb{R}^3\): node 1 displacement at time \(t\)

  • \(\underline{u}_2 \in \mathbb{R}^3\): node 2 displacement at time \(t\)

Force:

\[\begin{split}\underline{F} = \begin{bmatrix} \underline{f} \\ -\underline{f} \end{bmatrix} \,, \mathrm{where}\, \underline{F} \in \mathbb{R}^6\end{split}\]

and

\[\underline{f} = -k \frac{\underline{r} }{| \underline{r} |} \left( | \underline{r} | - \ell^\mathrm{us} \right) \,, \mathrm{where}\, \underline{r} = \underline{x}_2^0 + \underline{u}_2 - \underline{x}_1^0 - \underline{u}_1\]
\[\begin{split}\delta \underline{f} = \underline{\underline{K}}_t \begin{bmatrix} \delta \underline{u}_1\\ \delta \underline{u}_2 \end{bmatrix}\end{split}\]
\[\begin{split}\underline{\underline{K}}_t = \begin{bmatrix} \underline{\underline{A}} & -\underline{\underline{A}} \\ - \underline{\underline{A}} & \underline{\underline{A}} \end{bmatrix}\end{split}\]

and

\[\underline{\underline{A}} = k \left( \frac{\ell^\mathrm{us} }{|\underline{r} |} - 1\right) \underline{\underline{I}} - \frac{k \ell^\mathrm{us}}{|\underline{r}|^3}\left( \widetilde{r} \widetilde{r} + \underline{\underline{I}} (\underline{r}^T \underline{r} ) \right)\]

Aside: For \(\underline{a},\underline{r}\in \mathbb{R}^3\), \(\underline{r} (\underline{a}^T \underline{r}) = \left[ \widetilde{r} \widetilde{r} + \underline{\underline{I}} \left( \underline{r}^T \underline{r} \right) \right] \underline{a}\)
