Detailed Description

Functor to calculate gravity forces in a beam/mass element.

This struct serves as a function object to compute gravity forces for beam and rigid body elements.

Gravity force vector, FG = { {FG_1} = {m * g} {FG_2} = {m * eta_tilde * g} }

The forces are computed for each quadrature point (i_qp) of a given element (i_elem).
