Detailed Description

A linear systems solver for assembling and solving system matrices and constraints in OpenTurbine. This wraps Trilinos packages (Tpetra, Amesos2) for handling sparse matrix operations and linear systems solution.

This solver manages the assembly and solution of linear systems arising from the generalized-alpha based time integration of the dynamic structural problem. The linear systems include:

  • System stiffness matrix (K)
  • Tangent operator matrix (T)
  • Constraint matrices (B and B_t)
  • Combined system matrices for the complete structural problem
